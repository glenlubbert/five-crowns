# Kings Go Wild 👑

A free, browser-based five-suited rummy card game. Play against up to 6 AI opponents with adjustable skill levels.

**Play now:** https://kingsgowild.lubbert.com/

## How it works

11 rounds, dealing 3 to 13 cards. Each round, the card matching the hand size is wild (3s in round one… Kings in the final round), and Jokers are always wild. Arrange your whole hand into books and runs, go out, and force everyone else to count their leftovers. Lowest total score wins.

Built as a single HTML file — no dependencies, no build step.
